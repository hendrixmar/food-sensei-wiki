# Metabolismo

## Definición

El **metabolismo** es el conjunto de reacciones químicas que ocurren en las células del cuerpo para:
- Convertir alimentos en energía
- Construir y reparar tejidos
- Eliminar productos de desecho

## Dos Componentes

### Catabolismo
Procesos que **descomponen** moléculas para obtener energía:
- Digestión de alimentos
- Glucólisis (descomposición de glucosa)
- Oxidación de grasas

### Anabolismo
Procesos que **construyen** moléculas usando energía:
- Síntesis de proteínas
- Formación de tejidos
- Almacenamiento de glucógeno

## Metabolismo y Energía

### Tasa Metabólica
- **Basal (BMR):** Energía para funciones básicas en reposo
- **De actividad:** Energía para movimiento
- **Térmica:** Energía para digerir alimentos (TEF)

### Factores que Afectan el Metabolismo
- **Edad:** Disminuye con la edad
- **Masa muscular:** Más músculo = más metabolismo
- **Genética:** Diferencias individuales
- **Hormonas:** Tiroides, insulina, testosterona
- **Sueño:** La falta de sueño reduce el metabolismo
- **Ejercicio:** Lo aumenta

## Metabolismo en los Libros

### Frank Suarez
El metabolismo es la **clave** para la salud:
- Metabolismo disfuncionante = enfermedad
- Restaurar el metabolismo = curar enfermedad
- La Dieta 3x1 restaura el metabolismo

### David Perlmutter
El metabolismo cerebral depende de:
- Glucosa (pero no demasiada)
- Grasas (el cerebro es graso)
- Colesterol
- Cetonas (alternativa a glucosa)

### Jessie Inchauspé
El metabolismo de la glucosa es central:
- picos de glucosa = picos de insulina
- Insulina = almacenamiento de grasa
- El orden de alimentos afecta el metabolismo

## Termogénesis

Producción de calor del cuerpo, que consume energía.

### Tipos
1. **Adaptativa:** Respuesta al frío
2. **De alimentos (TEF):** Digerir comida
3. **Por actividad:** Ejercicio

### Implicaciones
- Algunos alimentos (picantes, cafeína) aumentan termogénesis
- La grasa parda produce calor
- Puede ayudar en pérdida de peso

## Síndrome Metabólico

Grupo de condiciones que ocurren juntas:
1. Obesidad abdominal
2. Glucosa elevada
3. Presión arterial alta
4. Colesterol anormal
5. Triglicéridos altos

Aumenta el riesgo de:
- Diabetes tipo 2
- Enfermedad cardíaca
- Stroke

## Síntesis

El metabolismo es:
- **Central** - Todo depende de él
- **Influenciable** - Se puede mejorar
- **Conectado** - Todos los sistemas están relacionados
- **Individual** - Cada persona es diferente
